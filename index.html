Wave Function Map Generator

About this generator

This demo uses constraint propagation to build a coherent map from simple rules.

  • Each cell begins with several possible tiles.
  • When a cell collapses to one tile, neighbors are pruned to keep things consistent.
  • Repeat until the whole grid forms natural‑looking structures.

Tile legend

  • Deep Blue: Deep water
  • Blue: Water
  • Yellow: Sand
  • Green: Grassland
  • Brown: Dirt
  • Dark Brown: Tree

Press Restart to see new variations.
